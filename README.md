# Mac Disk Usage Widget

Mac Disk Usage Widget is a lightweight macOS app that adds a native desktop widget for your system disk usage.

## What it shows
- Total storage
- Used storage
- Free storage
- Percentage used
- Progress bar in small, medium, and large widget sizes

## Highlights
- Native macOS look and feel (system fonts, native materials, light/dark support)
- Local-only disk stats (no remote services)
- No cleaners, no file scanning tools, no ads
- Manual refresh action available directly in the widget

## Requirements
- macOS 14 or newer

## Installation (current)
- Download the latest release artifact from the [Releases](../../releases) page.
- Open `MacDiskUsageWidget.app`.
- Add the widget from desktop widget gallery.

## Privacy
This app reads filesystem capacity information for the local system volume (`/`) using native macOS APIs only.
